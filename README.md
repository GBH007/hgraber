# HGraber

Граббер для манги.

## Возможности

1. Скачивать мангу с определенных сайтов
2. Просматривать скачанную мангу
3. Экспортировать мангу в формат для читалок

## Поддерживаемые БД

1. JSON
2. sqlite3
